LightNet now targets Astro v6.

## Breaking changes

- Sites using LightNet should upgrade their Astro app to v6.

## Migration

1. Update your site with:

```sh
npx @astrojs/upgrade
```

2. Follow the Astro v6 migration guide for any manual follow-up steps:

https://docs.astro.build/en/guides/upgrade-to/v6/
